## 🎯 **To Run Everything**

### **Frontend**:
```bash
cd frontend
npm run dev
# → http://localhost:3000
```

### **Backend** (when you have MongoDB):
```bash
cd backend
npm install
npm run dev
# → http://localhost:3001
```

### **MongoDB Setup**:
```bash
# Install MongoDB
brew install mongodb-community

# Start MongoDB
brew services start mongodb-community

# Or use MongoDB Atlas (cloud)
# https://www.mongodb.com/cloud/atlas
```
